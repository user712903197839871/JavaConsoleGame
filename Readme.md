# ⚓ MORSKOY boi: Console Edition ⚓
> [!IMPORTANT]
> Release 1.0

A high-performance, feature-rich Sea Battle (Battleship) game played entirely in the terminal. Built from scratch, featuring real-time input, advanced probability-based AI, and multiple game modes.

---

> [!CAUTION]
> Only supported on windows






---

## 🚀 Features
*   **Real-Time Controls**: Uses `jLine` library for a no-Enter-required gameplay.
*   **AI**: Includes an `Algorithm` that uses a probability matrix to hunt and destroy ships efficiently.
*   **Game Modes**:
    *   **Vanilla**: The classic experience.
    *   **Two Moves**: Double the fire-power per turn.
    *   **Sniper**: High-stakes accuracy mode.
*   **Customization**: Fully adjustable grid sizes, pointer graphics, and ANSI color schemes.
*   **PVP & PVE**: Play against a friend locally or challenge the computer.

> [!IMPORTANT]
> Big fonts and art gotten from:
> https://patorjk.com/software/taag/

## 🛠️ Installation & Execution

### Prerequisites
*   **Terminal Support**: A terminal that supports ANSI escape codes (VS Code Terminal, IntelliJ Terminal).
*   $\color{red}{\text{Windows operating system: Only supported on windows}}$


### How to Run
1.  Download `runable` folder from the root
2.  inside that folder there is a `MORSKOY boi.bat` file and a shotrcut
3.  ```double click .bat``` or the shotrcut to run the program
4.  Enjoy!

## 🎮 Controls
*   **`W | A | S | D`**: Navigate menus.
*   **`P`**: Select/Confirm choice.
*   **Coordinate Entry**: Enter coordinates (e.g., `A1`) when prompted during battle.
*  > [!Warning]
   > First comes the letter and then the number
*   **Other**: Other keybinds buring battle for advanced features.

## 🤖 Bot types
1.  **`Algorithmic Bot`**: Uses a specific algorithm to decide the positions of the ships.
2.  **`Random Bot`**: A dummy that makes random moves.
