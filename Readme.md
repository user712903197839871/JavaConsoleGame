# ⚓ MORSKOY boi: Console Edition ⚓
> [!IMPORTANT]
> Release 1.1

## ✨ About
Dive into an immersive, high-performance iteration of the classic naval strategy game, Sea Battle (Battleship), meticulously engineered from the ground up for a seamless terminal-based experience. This project transcends the simplicity of a standard command-line application, offering a feature-rich environment designed for both seasoned tacticians and casual players alike. <br>

The game boasts a robust architecture characterized by lightning-fast, real-time input handling that ensures instantaneous feedback with every tactical maneuver. At its core lies an advanced, sophisticated AI opponent, utilizing complex probability-based algorithms to simulate human-like decision-making. Instead of relying on randomized patterns, this AI anticipates your strategies, adjusts its tactics dynamically, and creates a challenging, unpredictable gameplay loop that demands genuine critical thinking. <br>

Furthermore, the platform provides diverse game modes, allowing users to tailor their experience in skirmishes. By stripping away graphical bloat, this terminal-centric design focuses entirely on the purity of competitive gameplay. It is an exploration of efficient software development where performance meets strategic depth, delivering an uncompromising, intellectually stimulating maritime warfare experience right within your favorite command-line interface. <br>

> [!CAUTION]
> Only supported on windows


## 🚀 Features
*   **Real-Time Controls**: Offers a no-Enter-required gameplay.
*   **AI**: Includes an `Algorithm` that hunts and destroys ships efficiently.
*   **Game Modes**:
    *   **Vanilla**: The classic experience.
    *   **Two Moves**: Double the fire-power per turn.
    *   **Sniper**: High-stakes accuracy mode.
*   **Customization**: Fully adjustable grid sizes, pointer graphics, and color schemes.
*   **PVP & PVE**: Play against a friend locally or challenge the computer.

## 🛠️ Installation & Execution

### Prerequisites
*   **Terminal Support**: A terminal that supports ANSI escape codes (VS Code Terminal, IntelliJ Terminal, powershell).
*   $\color{red}{\text{Windows operating system: Only supported on windows}}$


### How to Run
1.  Download the `SeaBattle` zip from the runable folder [The Runable](./runable)
2. Extract the files into a folder
3.  Inside that folder there is a `MORSKOY boi.bat` file and a shotrcut
4.  ```double click .bat``` or the shotrcut to run the program
5.  Enjoy!

## 🎮 Controls
*   **`W | A | S | D`**: Navigate menus.
*   **`P`**: Select/Confirm choice.
*   **Coordinate Entry**: Enter coordinates (e.g., `A1`) when prompted during battle.
> [!Warning]
> First comes the letter and then the number
*   **Other**: Other keybinds buring battle for advanced features.

## 🤖 Bot types
1.  **`Algorithmic Bot`**: Uses a specific algorithm to decide the positions of the ships.
2.  **`Random Bot`**: A dummy that makes random moves.
